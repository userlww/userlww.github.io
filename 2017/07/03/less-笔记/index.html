less是什么

   less是一门css预处理语言，扩展了css的功能，增加了变量，Mixin以及函数等特性，使得css更加易于维护，方便与制作主题与扩充。

less部分语法

变量

   例如

@bgcolor:#fff
#header{
  background:@bgcolor
  }

解析为

#header{
background:#fff
      }
