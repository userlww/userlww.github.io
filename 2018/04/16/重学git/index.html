重学git

2018-04-16 • 分类: 工具, git • 标签: git, 版本管理

开始上班以后，发现之前学的很多git相关的内容都忘记了，于是不得不重新学习了一遍，下面做一点简单的记录

常用命令整理

git add        将修改添加至暂存区
git commit     将修改提交到仓库
git clone      从远程克隆项目
git pull       将项目代码拉至本地
git push       向远程仓库提交
git status     查看工作区状态
git log        查看日志
git reset      版本回退
git init       创建版本库
git reflog     查看历史命令
git diff       查看版本库与缓存区的差别
git checkout --filename      撤销对文件的修改
git rm         删除文件
git remote add   建立远程连接
git branch     创建分支
git checkout   切换分支
git merge      合并分支
git branch -d   删除分支
