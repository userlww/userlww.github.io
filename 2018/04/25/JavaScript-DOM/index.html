2018-04-25 • 分类: 前端, JavaScript • 标签: DOM, JavaScript

DOM节点层次

DOM可以将HTML文档绘制成一个由多层次节点构成的结构，节点可以分为几种不同的类型，每一个节点都拥有各自的特点，数据和方法，也可能和其他节点存在某种关系，节点之间构成了层次，所有的页面表现为以一个特定节点为根节点的树形结构。文档节点是每个文档的根节点。HTML文档的根节点是html标签，文档节点是文档的最外层元素，每个文档只有一个根元素，HTML中的每一段标记都可以用一个节点表示，一共有12种节点，通过12个数值常量来表示

  • Node.ELEMENT_NODE(1);

  • Node.ATTRIBUTE_NODE(2)

  • Node.TEXT_NODE(3)

  • NODE.CDATA_SECTION_NODE(4)

    其余的不做罗列，需要的可以自行查阅文档，获取节点类型的方法，每一个节点都有一个nodeType的属性通过nodeType可以确定该节点的类型，例如

    if(someNode.nodeType==1){
     alert("Node is a element");
 }

   关于节点之间的关系，每一个节点都有一个childNodes属性，其中保存着一个NodeList对象，并且它也有length属性，但它并不是一个Array的实例，Nodelist对象会根据DOM的变化实时更新，可以通过方括号和item的方法来访问该节点的子节点，同时可以通过parentNode方法来访问节点的父节点，在同一个childList列表中的所有节点具有相同的父节点，他们互为同胞节点，可以通过previousSibing和nextSibing来访问它的上一个和下一个节点，第一个节点perviousSibing和最后一个节点的nextSibing的值都是null，父节点可以通过firstChild和lastChild来访问它的第一个和最后一个节点

操作节点

Dom提供了一些操作节点的方法，可以对节点进行一些操作，

  • 添加节点 appendChild()，用于向childList列表的最末尾添加一个节点，如果该节点已经存在，则将该节点移动到childList的最后一个位置，如果要将新节点添加到某一个特定的位置，可以使用insertBefore()方法，接受两个参数，要插入的节点和作为参照的节点，如果参照节点是null的话，则insertBefore和appendChild()执行的结果相同
  • 替换节点，replaceChild(),该方法接受两个参数，一个是要插入的节点，另一个是要替换的节点，该方法返回这个节点并将该节点从文档树中删除，同时要插入的节点将替换该节点的位置，被替换的节点从文档树删除，但是该节点存在于文档中，只是文档树中没有它的位置
  • 删除节点，removeChild()用于删除节点，接受的参数为要删除的节点，该方法同时将该节点返回
  • 复制节点，cloneNode(),用于创建一个和调用该方法的节点的一个相同的副本，接受一个布尔值的参数，如果为true，则复制该节点以及其整个子节点树，为false的时候只复制该节点，复制之后的文本节点属于文档，没有指定的父节点，除非通过appendChild，insertBefore()或者replaceChild()将它添加到文档中
  • normalize(),用于处理文档树中的文本节点，如果找到了空文本节点，则删除它，找到相邻的文本节点的话将它合并

Document类型

   document是window的一个实例,可以将其作为全局对象访问，通过它可以取得页面有关的信息，还能操作页面的外观和底层结构。

  • 通过documentElement属性指向页面中的html元素，
  • 通过childNodes列表来访问文档元素，
  • document.body可以获取页面中的body元素，
  • 通过document.doctype属性来访问文档类型信息，
  • 通过document.title访问title标签中的文本内容，
  • 通过document.url访问页面的url，
  • 通过document.domain访问页面的域名,由于跨域限制的问题，不同的子域的页面无法通过JavaScript通信。通过将每个页面的document。domain设置为相同的值，这样就可以互相访问对方包含的JavaScript对象
  • 通过document.getElementByTagName获取的到的元素存放在一个HTMLCollection对象之中，与NodeList对象，可以通过方括号或者item的方法来访问，可以通过nameItem的方法，在item中传入标签的name属性来获取对应标签
  • 使用document.implementation.hasFeature来确定浏览器实现了DOM的哪些部分
  • 使用document.write和document.writeIn向文档流中写入文本，write原样写入，writeIn会在末尾加入一个换行符（\n)
  • 使用querySelect和querySelectAll()来获取文档元素，括号中的参数类似于jQuery，前者获取符合查找条件的第一个元素，后者获取符合的所有元素，返回一个nodeList对象
  • 通过classList属性来对元素的所有类名进行操作
  • 通过document.activeElement属性来获取DOM焦点，元素获得焦点的方法有页面加载，用户输入以及调用focus()方法，文档刚刚加载完成时activeElement指向document.body,加载期间指向null，通过document.hasFocus()判断元素是否获取了焦点。
  • 通过document.readyState来获取文档加载的进度，值有loading和complete两种
  • document.head获取head标签
  • document.charset获取文档编码规则
  • 为元素添加自定义属性
  • innerHTML来获取调用元素的全部子节点
  • scrollIntoView()方法
  • children属性可以获得元素的所有同为元素节点的子节点
  • contains()方法用来判断一个节点是不是调用该方法的那个节点的子节点
  • CompareDocumentPosition()
  • offsetHeight:元素在垂直空间上占用的空间大小
  • offsetWidth:元素在水平方向上占用的空间大小
  • offsetLeft：元素的左外边框至包含元素的左内边框的距离
  • offsetTop：元素的上外边框至包含元素的上内边框的距离
  • offsetLeft和offsetTop属性与包含元素有关，包含元素的引用保存在offsetParent属性中
  • 元素的客户区大小是指元素内容及其内边距所占据的空间的大小，clientWidth和clientHeight属性表示元素内容区的宽高加上内边距的的宽高

DOM操作技术

动态脚本，使用appendChild方法向文档中插入script标签，并指定script脚本的src，type等属性值，可以加载外部的动态脚本。

动态样式 与动态脚本类似。可以通过appendChild的方法向文档中插入link标签来引入外部样式表
