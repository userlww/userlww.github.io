圣杯布局

2018-01-17 • 分类: 前端, CSS • 标签: CSS, 圣杯布局, 布局


   圣杯布局是2006年在国外提出的网页布局概念，在国内传说是玉伯引进的，这种布局方式有以下几点要求

  • 三列布局，中间宽度自适应，两侧定宽
  • 中间栏优先渲染
  • 允许任何列的高度最高

实现方法

HTML结构

<header>header</header>
<section class="container">
<div class="main">main</div>
<div class="left">left</div>
<div class="right">right</div>
</section>
<footer>footer</footer>

样式

*{
   margin: 0;
   padding: 0;
   color: white;
 }
 body
    {
        min-width: 700px;
    }
    header,footer
    {
        border: 1px solid red;
        background: #F55064;
        text-align: center;
    }
    .main,.left,.right{
        min-height: 500px;
        float: left;
    }
    .container
    {
        border: 1px solid yellow;
        overflow: hidden;             //清除浮动
        padding: 0 220px 0 200px;
    }
    .left
    {
        width: 200px;
        background: green;
        margin-left: -100%;          //将左边栏提到和中间栏同行显示
        position: relative;
        left: -200px;             //避免左边栏遮挡中间栏的内容
    }
    .right
    {
        width:220px;
        background:blue;
        margin-left:-220px;
        position: relative;
        right: -220px;
    }
    .main
    {
        background: black;
        width: 100%;
    }
footer
{
    clear: both;        //清除浮动
}
