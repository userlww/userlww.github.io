block, inline,inline-block对比

2018-01-08 • 分类: 前端, CSS • 标签: CSS, display


   今天撸页面的时候用到了display属性，看到了ws给的备选属性值，突然想inline-block和inline还有block到底有什么区别，于是翻了一下CSS权威指南，看了之后写一点自己的理解

  • inline ：属性值为inline表示以行内元素的形式显示，即不会独占一行，后面的行内元素将和该元素在同一行内显示，只有当容器的宽度不够使才会换行。
  • block：属性值为block表示以块状元素显示，无论容器的宽度是否足够。
  • inline-block：属性值为inline-block表示行内块元素，即元素本身以行内元素显示，元素的内容以块状元素显示。
