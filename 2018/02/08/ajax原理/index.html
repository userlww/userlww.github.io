为什么需要ajax

  传统的web网站，在提交完表单之后，浏览器需要重新加载整个页面，如果服务器长时间没能响应，客户端无法响应，那么用户体验将会非常差，当服务器端响应之后，浏览器需要加载整个页面，这样的话客户端的负担也会非常大，同时加载整个页面，需要接受大量数据女，影响网络的性能。

什么是ajax

  从本质上来说ajax=异步JavaScript+XML，是一种用于创建快速创建动态网页的技术，通过与后台进行少量的数据交换，实现网页的异步更新，可以在不加载整个网页的情况下,对于网页的某部分进行更新

原生js实现ajax

var xmlhttp=null; //声明一个变量，用来初始化XMLHttpRequest对象
if(window.XMLHttpRequest) {
    xmlhttp=new XMLHttpRequest();  //新版本的浏览器可以直接创建XMLHTTPRequest
}
else if(window.ActiveXObject){
    xmlhttp=new ActiveXObject("Microsoft XMLHTTP");
}
if(Xmlhttp!=null){
    xmlhttp.onreadystatechange=state_change;    //指定响应函数为state_change
    xmlhttp.open("GET","/example/xdom/note.xml",true);
//指定请求，这里要访问在/example/xdom/note.xml，true代表使用的是异步请求
    xmlhttp.send(null);
}
else{
    alert("Your browser doesn't support XMlHttP");
}

//创建响应函数state_change
function state_change(){
    if(xmlhttp.readyState==4){
        if(xmlhttp.status==200){
            //这里是响应函数的具体逻辑
        }
        else{
            alert("Problem retrieving XML data");
        }
    }

}
